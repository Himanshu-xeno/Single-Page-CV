# Single Page CV (HTML)

This project is a **Single Page CV built using semantic HTML**.
It focuses on structuring a resume webpage using proper HTML elements while also implementing **SEO meta tags, Open Graph tags, and a favicon**.

The goal of this project is to practice **clean HTML structure and web metadata fundamentals**.

---

## Project Features

* Semantic HTML structure
* Single-page CV layout
* Sections for:

  * Contact Information
  * Skills
  * Education
  * Experience
  * Social Links
* SEO meta tags for search engine optimization
* Open Graph tags for social media sharing
* Favicon support
* Clean structure ready for future CSS styling

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/Himanshu-xeno/single-page-cv.git
```

2. Navigate to the project folder

```
cd single-page-cv
```

3. Open the project

Simply open `index.html` in your browser.
